# Developer Onboarding

## Про проєкт
Цей проєкт є веб-застосунком для управління замовленнями в онлайн-магазині.

## Технології
- Frontend: HTML/CSS / React
- Backend: Node.js / Flask
- Database: PostgreSQL

## Структура проєкту
- architecture/ — архітектурні документи
- quality/ — документація тестування

## Як запустити документацію
1. Встановити MkDocs:
   pip install mkdocs

2. Запустити сервер:
   mkdocs serve

3. Відкрити:
   http://127.0.0.1:8000

## Рекомендації
- Дотримуватись структури документації
- Оновлювати SSD при зміні вимог
- Використовувати ідентифікатори (FR-01 тощо)
