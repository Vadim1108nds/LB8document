- Backend — бізнес-логіка  
- Database — зберігання даних  
- API — взаємодія між компонентами  

## 3. Взаємодія компонентів
Користувач взаємодіє з frontend, який надсилає запити до backend. Backend обробляє їх та взаємодіє з базою даних, після чого повертає результат користувачу.

## 4. API
- POST /login — авторизація  
- GET /products — отримання товарів  
- POST /orders — створення замовлення  
- PUT /orders/{id} — оновлення статусу
