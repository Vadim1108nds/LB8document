# Infrastructure Specification Document (ISD)

## 1. Середовище розгортання
Система розгортається у хмарному середовищі (AWS / Render) з використанням Docker.

## 2. Інфраструктура
- Backend сервер (Node.js / Flask)  
- PostgreSQL база даних  
- Nginx для балансування  

## 3. Масштабування
Система підтримує горизонтальне масштабування та CI/CD для автоматичного розгортання.
